# IA Reconnaissance Objet - YOLOv8

Projet de détection d’objets avec YOLOv8 et Streamlit.

## Fonctionnalités

- Upload d’image
- Détection d’objets
- Affichage des résultats

## Technologies

- Python
- YOLOv8
- Streamlit
- OpenCV

## Lancer le projet

```bash
pip install -r requirements.txt
streamlit run app.py
